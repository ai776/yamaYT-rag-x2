# yamaYT RAG仕訳インデックス

このファイルは `260428_yamaYT-rag-x-main` 専用のRAG仕訳表です。ほかのRAGリポジトリとは同期しません。

## 使い方

1. RAG検索やgrepの前に、必ずこのファイルを読む。
2. ユーザーテーマから主カテゴリを1つだけ決める。
3. 必要な場合だけ補助カテゴリを0〜1つ追加する。
4. 主カテゴリの「優先して読むファイル」から5〜8本を先に読む。
5. 足りない場合だけ `insights/` と `山本さん/` を追加grepする。
6. X記事に採用する具体事例は1〜2件、YouTube台本は2〜3件に絞る。
7. `insights/` の元ファイルは移動・改名しない。仕訳の正本はこのファイル。

## 共通ルール

- `insights/` を主RAGソースにする。
- `山本さん/` は補助ソース。ファイルは移動・改名しない。使用済みログやアーカイブ分けはしない。
- 1テーマに複数カテゴリを混ぜすぎない。主カテゴリ1つ、補助カテゴリ0〜1つまで。
- RAG外テーマは、RAGから無理に事実を補完しない。外部調査やユーザー提供情報を主にし、RAGは比喩・構成・話し方の参考に限定する。
